# Change: Create Personal Portfolio Site

## Why
构建一个专业的个人作品集网站，用于求职目的。目标用户为招聘方和技术同行，需要展示个人信息、经历、技能和作品。目前缺少一个统一的展示平台来突出个人能力和项目成果。

## What Changes
- 新增完整的个人作品集网站
- 包含个人信息展示页
- 经历和技能展示页
- 项目作品展示页
- 联系方式页面
- 响应式设计，支持移动端和桌面端
- 暗色/浅色模式切换
- SEO 优化配置

## Impact
- 新增网站：完整的个人作品集展示平台
- 影响技术栈：Next.js + TypeScript + Tailwind CSS
- 部署平台：Vercel
- 用户群体：招聘方和技术同行
- 验收标准：Lighthouse Performance/SEO > 95
