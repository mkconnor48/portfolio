## 📋 执行指南

### 测试顺序建议

1. **代码质量** → 2. **基础功能** → 3. **响应式** → 4. **交互** → 5. **无障碍** → 6. **性能**

### 时间估算

- **完整测试**: 2-3 小时
- **快速检查**: 30-45 分钟
- **性能专项**: 15-20 分钟

### 环境重置

每次测试前执行：

```bash
# 清理缓存
rm -rf .next
rm -rf node_modules/.cache

# 重新安装依赖
npm ci

# 启动干净的开发服务器
npm run dev
```
